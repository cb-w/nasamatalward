# متجر الهدايا والعطور

موقع ويب احترافي لمتجر هدايا وعطور مع لوحة تحكم كاملة لإدارة المنتجات واستقبال الطلبات.

## التشغيل

1. **تثبيت المتطلبات (مرة واحدة):**
   ```
   npm install
   ```

2. **تشغيل الموقع:**
   ```
   npm start
   ```

3. **افتح المتصفح:**
   - **المتجر (للزبائن):** http://localhost:3000
   - **لوحة التحكم:** http://localhost:3000/admin

4. **الدخول للوحة التحكم:**
   - كلمة المرور الافتراضية: `admin123`
   - يُنصح بتغييرها من تبويب "الإعدادات" بعد الدخول.

## ماذا يمكنك فعله؟

### من لوحة التحكم (/admin)
- **المنتجات:** إضافة منتجات جديدة، تعديل الاسم والسعر والوصف والفئة (عطور / هدايا)، رفع صورة، إخفاء أو إظهار منتج، حذف منتج.
- **الطلبات:** عرض كل الطلبات، تفاصيل كل طلب (العميل، الجوال، العنوان، المنتجات، المبلغ)، وتحديث حالة الطلب (جديد، قيد التحضير، تم الشحن، مكتمل، ملغي).
- **الإعدادات:** تغيير اسم المتجر وتغيير كلمة مرور لوحة التحكم.

### من المتجر (الصفحة الرئيسية)
- الزوار يشاهدون المنتجات، يضيفونها للسلة، ويكملون الطلب بإدخال الاسم والجوال والعنوان. الطلب يصل فوراً إلى لوحة التحكم.

## ملاحظات
- البيانات (المنتجات والطلبات) محفوظة في مجلد `data/` ولا تحتاج قاعدة بيانات خارجية.
- الصور المرفوعة تُحفظ في `public/uploads/`.
- العملة المعروضة: ريال سعودي (ر.س) — يمكن تغيير النص من الواجهة لاحقاً إذا أردت.
